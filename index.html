O que é HTML?

HTML é o componente básico da web, significa HiperText Markup Language (Linguaguem de Marcação de Hipertexto), foi criado pelo britânico Tim Berners-Lee.

Permite inserir o conteúdo e estabelecer a estrutura básica de um website, portanto serve para dar significado e organizar as informações de uma página web. Sem o HTML, o navegador não conseguiria exibir textos, carregar imagens e outros conteúdos.

Portanto, podemos considerar o HTML o “esqueleto” da sua página.

Como funciona o HTML?

Através de um documento HTML, ou seja, um documento com a extensão .html ou.htm., o navegador faz a leitura do arquivo e renderiza o seu conteúdo para que o usuário final possa visualizá-lo. Os arquivos .html podem ser visualizados em qualquer navegador (como Google Chrome, Safari, Mozilla Firefox ou Internet Explorer).

O que são tags?

O documento HTML consiste em uma série de tags. As tags são códigos que definem toda a estrutura da página, tais como o seu tamanho, a fonte da letra, as cores, as quebras de linha e etc. A maioria dos elementos do documento HTML são compostos por uma estrutura de abertura e uma de fachamento, como e . Principais tags do HTML

Atualmente existem mais de 140 tags, mas algumas delas quase não são utilizadas. Dentre as mais utilizadas temos:

  1. <head> - local para declarar todas informações, como título e metadados da sua página;
  2. <title> - define o título da página;
  3. <body> - local para declarar todos os elementos que irão compor o corpo da página;
  4. <h1> ... <h6> - tags para definir um título e subtítulos, variando de 1 a 6, sendo h1 o título mais importante e h6 o de menor importância;
  5. <p> - tag para definir um parágrafo;
  6. <a> - tag de link, junto ao atributo href="" é responsável pela principal característica da web;
  7. <header> - define um cabeçalho;
  8. <section> - define uma seção;
  9. <article> - define um artigo;
  10. <div> - define uma divisão;
  11. <footer> - define um rodapé;
  12. <nav> - define uma área de navegação (como menus);
  13. <table> - define uma tabela;
  14. <ol> - define uma lista ordenada;
  15. <ul> - define uma lista não ordenada;
  16. <li> - define o item de uma lista;
  17. <form> - define um formulário;
  18. <input> - define os campos do formulário;
  19. <textarea> - define uma área para o usuário digitar um texto;
  20. <button> - define um botão;
  21. <img> - permite inserir uma imagem no seu documento;
  22. <font> - tag não mais suportada no HTML5
